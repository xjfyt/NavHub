## 6、版本规划

### 6.1 版本号规则

采用语义化版本号：`主版本号.次版本号.修订号`

- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正

### 6.2 发布周期

- 修订版：按需发布
- 次版本：每月发布
- 主版本：每季度发布
