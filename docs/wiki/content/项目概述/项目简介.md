## 1、NavHub 是什么

NavHub 是一款**开源、自托管的个人导航主页**，适合部署在家庭实验室、内网或 VPS 上。它能够：

- 提供多种图标样式（方形、长条、大格、圆形），支持自定义颜色、字体、对齐
- 内置全面接入 Iconify 集合的图标库，支持多数据源管理、前端模糊搜索
- 支持文件夹功能（2×2 / 3×3 网格预览，拖拽排序与合并）
- 提供小组件系统（时钟、天气、搜索等，三档固定尺寸自适应）
- 内置壁纸库（Bing / Wikimedia / NASA / Unsplash / Wallhaven / Pexels / Pixabay 多来源）
- 支持分类导航（多分类侧栏，拖拽跨分类移动）
- 支持 SSO 单点登录（Casdoor / Authentik 等 OIDC 兼容）
- 支持多用户多租户数据隔离，超级管理员后台
- 支持 S3 存储（图片上传及系统托管基于 S3 兼容的对象存储）

## 3、核心能力一览

| 能力 | 说明 |
|------|------|
| **图标管理** | 多种样式、Iconify 集合、拖拽排序 |
| **分类系统** | 多分类侧栏、跨分类移动、推送分类 |
| **小组件** | 时钟、天气、搜索等，自适应尺寸 |
| **壁纸库** | 多来源、随机轮换、响应式网格适配 |
| **用户系统** | 多租户、角色权限、SSO 单点登录 |
| **存储系统** | S3 兼容对象存储、CDN 加速 |
| **部署方式** | Docker 容器化、一键部署 |

## 4、技术架构

- **后端**：Rust (Axum + SQLx + Redis)
- **前端**：React 18 + TypeScript + Vite + @dnd-kit
- **数据库**：PostgreSQL 15
- **缓存**：Redis 7
- **认证**：Casdoor OIDC（所有人）+ 账号密码（仅超级管理员）
- **存储**：PostgreSQL（主数据）+ Redis（session、state、缓存）+ S3（文件存储）

## 5、适用场景

### 5.1 个人用户

- **家庭实验室导航**：将所有自建服务（NAS、路由器、监控面板等）集中管理
- **开发环境快捷入口**：本地开发时快速访问各种开发工具和文档
- **个人书签管理**：替代浏览器书签，支持分类和图标自定义

### 5.2 团队使用

- **内网服务门户**：团队内部服务（GitLab、Jenkins、Wiki等）统一入口
- **多用户权限管理**：不同角色看到不同的导航内容
- **SSO 集成**：与现有认证系统（Casdoor、Authentik等）无缝对接

### 5.3 运维场景

- **自托管优先**：数据完全自主掌控，无第三方依赖
- **容器化部署**：支持 Docker 一键部署，易于维护和升级
- **轻量级架构**：单进程设计，资源占用低
