# 三、样式系统

> **对应代码**：`frontend/src/styles.css`、`frontend/src/shell.css`、`frontend/src/hooks/useColorMode.ts`
> **维护提示**：新增/修改 CSS 变量、类名规范或主题系统时同步更新本文档。

## 1、样式架构概述

NavHub 采用 **纯 CSS + CSS 变量** 方案，不使用 Tailwind CSS 或 CSS-in-JS。样式文件分为两层：

| 文件 | 职责 | 类名前缀 |
|------|------|----------|
| `styles.css` | 全局样式、设计令牌、组件样式 | `.tile-*`、`.modal-*`、`.widget-*` |
| `shell.css` | 外壳布局、登录页、管理后台 | `.nh-*`（namespace 避免冲突） |

## 2、设计令牌（Design Tokens）

## 3、类名命名规范

### （1）BEM-like 命名

采用 BEM 风格但简化分隔符：

```
.block                    → 组件块
.block-element            → 组件元素
.block--modifier          → 修饰符
```

**示例**：

```css
/* 块 */
.tile { }

/* 元素 */
.tile-header { }
.tile-body { }
.tile-icon { }

/* 修饰符 */
.tile--selected { }
.tile--dragging { }
.tile--readonly { }
```

### （2）命名空间前缀

`shell.css` 中的样式使用 `nh-` 前缀避免与全局样式冲突：

```css
/* shell.css */
.nh-boot { }
.nh-boot-spinner { }
.nh-login-overlay { }
.nh-login-card { }
.nh-side { }
.nh-ws { }
```

## 4、玻璃态效果（Glassmorphism）

NavHub 的视觉风格以玻璃态为核心，关键 CSS 模式：

```css
/* 标准玻璃卡片 */
.glass-card {
  background: linear-gradient(
    180deg,
    rgba(255, 255, 255, 0.14),
    rgba(255, 255, 255, 0.08)
  );
  border: 1px solid rgba(255, 255, 255, 0.16);
  border-radius: 24px;
  backdrop-filter: blur(28px) saturate(140%);
  -webkit-backdrop-filter: blur(28px) saturate(140%);
  box-shadow:
    0 24px 60px rgba(0, 0, 0, 0.45),
    0 1px 0 rgba(255, 255, 255, 0.18) inset,
    0 0 0 0.5px rgba(255, 255, 255, 0.06) inset;
}
```

**关键属性**：

| 属性 | 值 | 作用 |
|------|----|------|
| `backdrop-filter` | `blur(28px) saturate(140%)` | 背景模糊 + 饱和度增强 |
| `background` | 渐变半透明白 | 玻璃质感 |
| `box-shadow` | 多层阴影 | 深度 + 内发光 |

## 5、主题系统

### （1）色深模式（Dark/Light）

通过 `data-mode` 属性切换：

```typescript
// hooks/useColorMode.ts
export function useColorMode(mode: string | undefined) {
  useEffect(() => {
    const root = document.documentElement;
    const actualMode = mode || "auto";
    const mm = window.matchMedia("(prefers-color-scheme: light)");
    
    const applyMode = () => {
      const actual = actualMode === "auto"
        ? (mm.matches ? "light" : "dark")
        : actualMode;
      root.dataset.mode = actual;
    };
    
    applyMode();
    if (actualMode === "auto") {
      mm.addEventListener("change", applyMode);
      return () => mm.removeEventListener("change", applyMode);
    }
  }, [mode]);
}
```

```css
/* 根据模式调整颜色 */
[data-mode="light"] {
  --text: #1a1a1a;
  --text-soft: rgba(0, 0, 0, 0.76);
  --glass-bg: rgba(255, 255, 255, 0.6);
  --glass-border: rgba(0, 0, 0, 0.1);
}
```

### （2）主题色（Theme）

通过 CSS 变量覆盖实现不同主题：

```css
/* styles.css - 主题定义 */
[data-theme="dawn"] {
  --accent: #ffd7a5;
  --bg-gradient: linear-gradient(135deg, #1a1a2e, #16213e);
}

[data-theme="ocean"] {
  --accent: #64b5f6;
  --bg-gradient: linear-gradient(135deg, #0d1b2a, #1b2838);
}

[data-theme="forest"] {
  --accent: #81c784;
  --bg-gradient: linear-gradient(135deg, #1b3a2d, #2d4a3e);
}
```

## 6、响应式布局

### （1）断点策略

不使用固定断点，而是基于容器宽度自适应：

```css
/* 图标网格：自动填充 */
.icon-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(112px, 1fr));
  gap: 20px 16px;
}

/* 组件网格：固定列数（可配置） */
.widget-grid {
  display: grid;
  grid-template-columns: repeat(var(--grid-cols, 6), minmax(0, 1fr));
  gap: 14px;
}
```

### （2）移动端适配

```css
/* 侧边栏响应式 */
.app {
  display: grid;
  grid-template-columns: auto 1fr;
  gap: var(--sidebar-gap);
}

/* 窄屏：侧边栏变为抽屉 */
@media (max-width: 768px) {
  .app {
    grid-template-columns: 1fr;
  }
  
  .sidebar {
    position: fixed;
    left: 0;
    top: 0;
    bottom: 0;
    z-index: 100;
    transform: translateX(-100%);
    transition: transform 0.3s var(--smooth);
  }
  
  .mobile-nav-open .sidebar {
    transform: translateX(0);
  }
  
  .mobile-nav-backdrop {
    position: fixed;
    inset: 0;
    background: rgba(0, 0, 0, 0.5);
    z-index: 99;
    opacity: 0;
    pointer-events: none;
    transition: opacity 0.3s;
  }
  
  .mobile-nav-open .mobile-nav-backdrop {
    opacity: 1;
    pointer-events: auto;
  }
}
```

## 7、动画规范

### （1）动画曲线

```css
:root {
  /* 弹性曲线（按钮、卡片） */
  --spring: cubic-bezier(0.34, 1.56, 0.64, 1);
  
  /* 平滑曲线（过渡、淡入） */
  --smooth: cubic-bezier(0.25, 0.46, 0.45, 0.94);
}
```

### （2）标准动画模式

```css
/* 淡入 */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* 弹入 */
@keyframes scaleIn {
  from {
    opacity: 0;
    transform: translateY(12px) scale(0.96);
  }
  to {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}

/* 旋转（加载） */
@keyframes spin {
  to { transform: rotate(360deg); }
}
```

### （3）过渡规范

| 元素 | 属性 | 时长 | 曲线 |
|------|------|------|------|
| 按钮 hover | `transform`, `box-shadow` | 150ms | `ease` |
| 卡片 hover | `box-shadow` | 200ms | `var(--smooth)` |
| 模态框 | `opacity`, `transform` | 280ms | `var(--spring)` |
| 侧边栏 | `transform` | 300ms | `var(--smooth)` |

## 8、滚动条隐藏

全局隐藏滚动条，保留滚动功能：

```css
/* WebKit 浏览器 */
*::-webkit-scrollbar {
  width: 0;
  height: 0;
  display: none;
}

/* Firefox */
* {
  scrollbar-width: none;
}

/* IE */
* {
  -ms-overflow-style: none;
}
```

## 9、无障碍（A11Y）

### （1）焦点管理

```css
/* 鼠标点击时隐藏默认 outline */
input:focus:not(:focus-visible),
textarea:focus:not(:focus-visible) {
  outline: none;
}

/* 键盘 Tab 时显示焦点环 */
:focus-visible {
  outline: 2px solid var(--accent);
  outline-offset: 2px;
}
```

### （2）减少动画

```css
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```
