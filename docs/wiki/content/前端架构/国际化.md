# 四、国际化

> **对应代码**：`frontend/src/i18n/index.tsx`、`frontend/src/i18n/translate.ts`、`frontend/src/i18n/dictionaries.ts`
> **维护提示**：新增/修改翻译键、支持新语言或调整 i18n 架构时同步更新本文档。

## 1、国际化概述

NavHub 采用 **自研轻量 i18n 方案**，不依赖 `react-i18next`、`react-intl` 等外部库。核心特性：

| 特性 | 实现 |
|------|------|
| 零依赖 | 纯 React Context + 纯函数 |
| 类型安全 | TypeScript 类型约束语言代码 |
| 插值支持 | `{name}` 占位符语法 |
| 回退链 | 当前语言 → 中文 → key 本身 |
| 持久化 | `localStorage` 存储语言偏好 |
| 自动探测 | `navigator.language` → 中文/英文 |

## 2、架构设计

### （1）三层分离

```
┌─────────────────────────────────────────────┐
│  React 层 (index.tsx)                        │
│  ├─ I18nProvider (Context)                   │
│  └─ useI18n() Hook                           │
├─────────────────────────────────────────────┤
│  纯逻辑层 (translate.ts)                     │
│  ├─ translate() 查表 + 插值                   │
│  ├─ detectLang() 语言探测                     │
│  └─ interpolate() 占位符替换                  │
├─────────────────────────────────────────────┤
│  数据层 (dictionaries.ts)                    │
│  └─ dictionaries: { zh: {...}, en: {...} }   │
└─────────────────────────────────────────────┘
```

### （2）类型定义

```typescript
// i18n/translate.ts
export type Lang = "zh" | "en";
export const LANGS: Lang[] = ["zh", "en"];
export type Dict = Record<string, string>;
export type Dictionaries = Record<Lang, Dict>;
export type TParams = Record<string, string | number>;
```

## 3、核心实现

### （1）翻译函数

纯函数，便于单元测试：

```typescript
// i18n/translate.ts
export function translate(
  dicts: Dictionaries,
  lang: Lang,
  key: string,
  params?: TParams,
  fallbackLang: Lang = "zh",
): string {
  // 1. 尝试当前语言
  const primary = dicts[lang];
  if (primary && Object.prototype.hasOwnProperty.call(primary, key)) {
    return interpolate(primary[key], params);
  }
  
  // 2. 回退到中文
  const fb = dicts[fallbackLang];
  if (fb && Object.prototype.hasOwnProperty.call(fb, key)) {
    return interpolate(fb[key], params);
  }
  
  // 3. 未命中 → 返回 key（便于发现漏翻）
  return key;
}
```

### （2）插值函数

```typescript
// i18n/translate.ts
const PLACEHOLDER = /\{(\w+)\}/g;

function interpolate(template: string, params?: TParams): string {
  if (!params) return template;
  return template.replace(PLACEHOLDER, (whole, name: string) => {
    const v = params[name];
    return v === undefined || v === null ? whole : String(v);
  });
}
```

**使用示例**：

```typescript
t("welcome.message", { name: "张三" })
// "欢迎回来，张三"

t("items.count", { count: 5 })
// "共 5 个项目"
```

### （3）语言探测

```typescript
// i18n/translate.ts
export function detectLang(input: {
  stored: string | null | undefined;
  navigator: string | null | undefined;
}): Lang {
  // 1. 优先使用用户偏好（localStorage）
  const stored = input.stored?.toLowerCase();
  if (stored && LANGS.includes(stored as Lang)) {
    return stored as Lang;
  }
  
  // 2. 探测浏览器语言
  const nav = (input.navigator || "").toLowerCase();
  if (nav.startsWith("zh")) return "zh";
  
  // 3. 默认英文
  return "en";
}
```

## 4、React 集成

### （1）I18nProvider

```typescript
// i18n/index.tsx
export function I18nProvider({ children }: { children: ReactNode }) {
  const [lang, setLangState] = useState<Lang>(initialLang);
  
  const setLang = useCallback((next: Lang) => {
    setLangState(next);
    persistLang(next); // 写入 localStorage
  }, []);
  
  const t = useCallback(
    (key: string, params?: TParams) => translate(dictionaries, lang, key, params),
    [lang],
  );
  
  const value = useMemo<I18nContextValue>(
    () => ({ lang, setLang, t }),
    [lang, setLang, t],
  );
  
  return <I18nContext.Provider value={value}>{children}</I18nContext.Value>;
}
```

### （2）useI18n Hook

```typescript
// i18n/index.tsx
export function useI18n(): I18nContextValue {
  const ctx = useContext(I18nContext);
  if (!ctx) {
    throw new Error("useI18n must be used within <I18nProvider>");
  }
  return ctx;
}
```

### （3）组件中使用

```typescript
// components/Shell.tsx
import { useI18n } from "../i18n";

export const Shell = () => {
  const { t, lang, setLang } = useI18n();
  
  return (
    <div>
      <span>{t("guest.banner")}</span>
      <button>{t("common.login")}</button>
      
      {/* 语言切换 */}

        <option value="zh">{t("lang.zh")}</option>
```

### （2）Key 命名规范

采用 `域.具体含义` 的层级结构：

| 域 | 说明 | 示例 |
|----|------|------|
| `common.*` | 通用按钮/词汇 | `common.save`、`common.cancel` |
| `login.*` | 登录页 | `login.submit`、`login.err.*` |
| `guest.*` | 访客相关 | `guest.banner` |
| `nav.*` | 导航/空状态 | `nav.empty.noGroupsTitle` |
| `settings.*` | 设置面板 | `settings.language` |
| `lang.*` | 语言名称 | `lang.zh`、`lang.en` |
| `widget.*` | 组件名称 | `widget.clock`、`widget.weather` |

## 7、持久化策略

```typescript
// i18n/index.tsx
const LANG_STORAGE_KEY = "navhub_lang";

function readStoredLang(): string | null {
  try {
    return window.localStorage.getItem(LANG_STORAGE_KEY);
  } catch {
    return null; // 隐私模式等
  }
}

function persistLang(lang: Lang) {
  try {
    window.localStorage.setItem(LANG_STORAGE_KEY, lang);
  } catch {
    /* 忽略 */
  }
}
```

## 9、最佳实践

### （1）新增翻译键

1. 在 `dictionaries.ts` 的 `zh` 中添加中文文案
2. 在 `en` 中添加英文翻译（可渐进补全）
3. 使用 `t("domain.key")` 调用

### （2）插值参数

## 10、测试

### （1）单元测试

```typescript
// i18n/translate.test.ts
import { translate, detectLang, interpolate } from "./translate";

describe("translate", () => {
  const dicts = {
    zh: { "test.key": "测试 {name}" },
    en: { "test.key": "Test {name}" },
  };
  
  it("returns translated string for current lang", () => {
    expect(translate(dicts, "zh", "test.key", { name: "A" })).toBe("测试 A");
  });
  
  it("falls back to zh when key missing in current lang", () => {
    expect(translate(dicts, "en", "missing.key")).toBe("missing.key");
  });
});

describe("detectLang", () => {
  it("prefers stored preference", () => {
    expect(detectLang({ stored: "en", navigator: "zh-CN" })).toBe("en");
  });
  
  it("detects zh from navigator", () => {
    expect(detectLang({ stored: null, navigator: "zh-CN" })).toBe("zh");
  });
  
  it("defaults to en", () => {
    expect(detectLang({ stored: null, navigator: "fr" })).toBe("en");
  });
});
```
