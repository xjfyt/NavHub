# 路由设计

## 概述
NavHub 使用 Axum 框架的路由系统，所有 API 路由集中在 backend/src/routes.rs 中定义。

## 路由层次

### 公开路由（无需认证）
```
GET  /auth/login              # SSO 登录重定向
GET  /auth/callback           # OIDC 回调
POST /auth/password           # 密码登录
POST /auth/logout             # 用户登出
GET  /auth/status             # 认证状态
GET  /api/config/public       # 公开配置
GET  /api/healthz             # 健康检查
GET  /api/readyz              # 就绪检查
```

### 游客可访问路由（可选认证）
```
GET  /api/workspace           # 工作区数据
GET  /api/wallpapers          # 壁纸列表
GET  /api/wallpaper-sources   # 壁纸源列表
GET  /api/favicon             # Favicon 代理
GET  /api/favicon/search      # Favicon 搜索
```

### 用户路由（需认证）
```
GET/PATCH /api/me                     # 用户信息
GET/PATCH /api/me/preferences         # 用户偏好
GET/POST  /api/me/engines             # 自定义搜索引擎
DELETE    /api/me/engines/:id         # 删除搜索引擎
GET       /api/me/messages            # 消息列表
POST      /api/me/messages/read-all   # 标记全部已读
POST      /api/me/messages/:id/read   # 标记单条已读
GET       /api/me/export              # 数据导出
```

### 分类/图标/组件 CRUD
```
POST   /api/groups                    # 创建分类
PATCH  /api/groups/:id                # 更新分类
DELETE /api/groups/:id                # 删除分类
POST   /api/groups/reorder            # 分类排序
POST   /api/groups/:id/reorder-items  # 分类内项目排序

POST   /api/icons                     # 创建图标
PATCH  /api/icons/:id                 # 更新图标
DELETE /api/icons/:id                 # 删除图标
POST   /api/icons/:id/merge-into/:target  # 合并为文件夹
POST   /api/icons/:id/extract-item/:target  # 从文件夹提取
POST   /api/icons/reorder             # 图标排序

POST   /api/widgets                   # 创建组件
PATCH  /api/widgets/:id               # 更新组件
DELETE /api/widgets/:id               # 删除组件
```

### 上传
```
POST   /api/upload                    # 文件上传 (multipart)
```

### 小组件数据
```
GET    /api/widgets/weather           # 天气数据
GET    /api/widgets/hot               # 热搜数据
GET    /api/widgets/music/search      # 音乐搜索
GET    /api/widgets/music/song/:id    # 音乐详情
```

### 管理后台路由
```
GET    /api/admin/dashboard           # 仪表盘
GET    /api/admin/users               # 用户列表
PATCH  /api/admin/users/:id           # 更新用户
DELETE /api/admin/users/:id           # 删除用户
POST   /api/admin/groups/:id/push     # 推送分类
DELETE /api/admin/groups/:id/push     # 取消推送
GET    /api/admin/audit               # 审计日志
GET/PATCH /api/admin/settings         # 系统设置
GET/PATCH /api/admin/sso              # SSO 配置
```

### 静态资源
```
GET    /uploads/*                     # 用户上传文件
GET    /*                             # SPA fallback
```

## 路由注册方式

```rust
let api = Router::new()
    .route("/me", get(handlers::me::get_me).patch(handlers::me::patch_me))
    .route("/groups", post(handlers::groups::create))
    // ...
    .layer(axum::middleware::from_fn_with_state(
        state.clone(),
        auth::middleware::require_login,
    ));
```

## 中间件应用

- 游客路由：optional_login 中间件
- 用户路由：require_login 中间件
- 密码登录：password_login_limit 限流中间件
- 全局：Cache-Control: no-store 响应头
