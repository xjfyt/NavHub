# 样式规范

## 概述
NavHub 使用 CSS 变量 + 原生 CSS 实现样式系统，不依赖 Tailwind CSS 工具类。

## 设计 Token

### 颜色系统
```css
:root {
  --glass-blur: 28px;
  --glass-bg: rgba(255, 255, 255, 0.08);
  --glass-bg-strong: rgba(255, 255, 255, 0.15);
  --glass-border: rgba(255, 255, 255, 0.2);
  --glass-border-soft: rgba(255, 255, 255, 0.1);
  --text: #ffffff;
  --text-soft: rgba(255, 255, 255, 0.76);
  --text-mute: rgba(255, 255, 255, 0.5);
  --accent: #ffd7a5;
  --danger: #ff8a8a;
  --ok: #9be7b4;
}
```

### 字体系统
```css
:root {
  --font-sans: -apple-system, BlinkMacSystemFont, "PingFang SC",
    "Microsoft YaHei", "Hiragino Sans GB", "Helvetica Neue",
    "Segoe UI", system-ui, sans-serif;
  --font-mono: ui-monospace, "SF Mono", "Menlo", "Cascadia Code",
    "Roboto Mono", "Source Code Pro", Consolas, monospace;
}
```

### 圆角系统
```css
:root {
  --radius-sm: 10px;
  --radius-md: 16px;
  --radius-lg: 24px;
  --radius-xl: 30px;
}
```

### 动画曲线
```css
:root {
  --spring: cubic-bezier(0.34, 1.56, 0.64, 1);
  --smooth: cubic-bezier(0.25, 0.46, 0.45, 0.94);
}
```

### 阴影系统
```css
:root {
  --tile-shadow: 0 3px 12px rgba(0, 0, 0, 0.22),
    0 1px 0 rgba(255, 255, 255, 0.24) inset,
    0 0 0 0.5px rgba(0, 0, 0, 0.1) inset;
  --tile-shadow-hover: 0 12px 32px rgba(0, 0, 0, 0.32),
    0 4px 8px rgba(0, 0, 0, 0.12),
    0 1px 0 rgba(255, 255, 255, 0.3) inset;
}
```

### 字号系统
```css
:root {
  --font-xs: 10px;
  --font-sm: 12px;
  --font-base: 14px;
  --font-lg: 16px;
  --font-xl: 28px;
}
```

### 间距系统（8px 基础）
```css
:root {
  --sp-1: 4px;
  --sp-2: 8px;
  --sp-3: 12px;
  --sp-4: 16px;
  --sp-6: 24px;
}
```

## 命名规范

### BEM 命名
- 块：`.nh-shell`、`.nh-sidebar`
- 元素：`.nh-shell__header`
- 修饰符：`.nh-shell--compact`

### 命名空间
所有类名使用 `nh-` 前缀，避免与设计稿样式冲突。

## 玻璃拟态效果

NavHub 大量使用玻璃拟态（Glassmorphism）效果：

```css
.nh-glass {
  background: var(--glass-bg);
  backdrop-filter: blur(var(--glass-blur));
  -webkit-backdrop-filter: blur(var(--glass-blur));
  border: 1px solid var(--glass-border);
  box-shadow: var(--glass-shadow);
}
```

## 响应式设计

主要断点：
- 移动端：< 768px
- 平板：768px - 1024px
- 桌面：> 1024px

## 文件组织

- styles.css：全局样式和设计 Token
- shell.css：外壳组件样式（nh- 命名空间）
- 组件内联样式：用于动态样式和条件样式
