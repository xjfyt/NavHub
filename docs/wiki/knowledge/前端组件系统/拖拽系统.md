# 拖拽系统实现

## 概述
NavHub 使用 @dnd-kit 库实现图标的拖拽排序和文件夹合并功能。

## 核心功能

### 1. 图标拖拽排序
- 同一分类内的图标拖拽排序
- 跨分类拖拽移动图标
- 拖拽到文件夹合并

### 2. 分类拖拽排序
- 侧边栏分类顺序调整
- 普通用户保存到 sidebar_order
- 管理员保存到全局 sort_order

### 3. 小组件拖拽
- 小组件位置调整
- 小组件宽度调整

## 技术实现

### DndContext 配置
```tsx
import { DndContext, DragOverlay } from '@dnd-kit/core';
import { SortableContext, verticalListSortingStrategy } from '@dnd-kit/sortable';

function NavView({ icons, onReorder }) {
  return (
    <DndContext
      sensors={sensors}
      collisionDetection={closestCenter}
      onDragStart={handleDragStart}
      onDragEnd={handleDragEnd}
    >
      <SortableContext items={icons} strategy={verticalListSortingStrategy}>
        {icons.map(icon => <SortableIcon key={icon.id} icon={icon} />)}
      </SortableContext>
      <DragOverlay>
        {activeIcon ? <IconTile icon={activeIcon} /> : null}
      </DragOverlay>
    </DndContext>
  );
}
```

### Sortable 组件
```tsx
import { useSortable } from '@dnd-kit/sortable';
import { CSS } from '@dnd-kit/utilities';

function SortableIcon({ icon }) {
  const { attributes, listeners, setNodeRef, transform, transition } =
    useSortable({ id: icon.id });

  const style = {
    transform: CSS.Transform.toString(transform),
    transition,
  };

  return (
    <div ref={setNodeRef} style={style} {...attributes} {...listeners}>
      <IconTile icon={icon} />
    </div>
  );
}
```

## API 交互

### 图标排序
```typescript
// POST /api/icons/reorder
{ groupId: "xxx", order: ["icon1", "icon2", "icon3"] }
```

### 分类排序
```typescript
// POST /api/groups/reorder
{ order: ["group1", "group2", "group3"] }
```

### 图标合并为文件夹
```typescript
// POST /api/icons/:id/merge-into/:target
```

## 权限控制

- 推送分类对普通用户只读（readOnly: true）
- 只有所有者或管理员可以编辑
- 拖拽前检查权限，禁用不可拖拽项

## 视觉反馈

- 拖拽时显示半透明预览
- 拖拽到目标区域高亮
- 排序动画平滑过渡
- 使用 spring 动画曲线

## 性能优化

- 虚拟列表：大量图标时使用虚拟滚动
- 延迟更新：拖拽结束才发送 API 请求
- 乐观更新：先更新 UI 再同步后端
